# Посібник користувача: VR Тіла обертання

Вітаємо у додатку для вивчення стереометрії! Цей інструмент допоможе вам краще зрозуміти, як утворюються тіла обертання.

## Як користуватися додатком:

1. **Вибір фігури:** У верхній частині панелі керування (зліва) виберіть тип тіла обертання зі списку. Доступні варіанти:
   - Циліндр (утворюється обертанням прямокутника).
   - Конус (утворюється обертанням прямокутного трикутника).
   - Еліпсоїд / Куля (утворюється обертанням півеліпса або півкола).

2. **Налаштування розмірів:**
   - **Радіус основи (r):** введіть числове значення. Фігура миттєво змінить свою ширину.
   - **Висота осі обертання (h):** введіть числове значення. Фігура миттєво змінить свою висоту. *(Зверніть увагу: якщо радіус та висота осі для еліпсоїда рівні, ви отримаєте ідеальну кулю).*

3. **Дослідження утворення фігури:**
   - Використовуйте **повзунок "Кут обертання"**, щоб вручну розгортати фігуру. Ви побачите червону плоску фігуру, яка обертається навколо чорної осі і залишає за собою синій напівпрозорий слід (саме тіло).
   - Натисніть кнопку **"▶ Анімувати"**, щоб запустити автоматичне обертання фігури на 360 градусів. Ви можете поставити процес на паузу в будь-який момент.

4. **Результати обчислень:**
   У нижній частині панелі автоматично виводяться розраховані значення Об'єму (V) та Площі поверхні (S) повної фігури на основі ваших розмірів.
