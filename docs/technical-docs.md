# Технічна документація

## Архітектура проєкту
Проєкт реалізовано як клієнтський web-додаток без використання серверної частини.
- `index.html` — лендінг проєкту з інформацією та посиланнями.
- `demo.html` — основна сторінка з 3D-сценою та HTML-інтерфейсом.
- `app.js` — логіка застосунку.

## Математичні моделі обчислень
У файлі `app.js` реалізовано функцію `calculateMath(type, r, h)`, яка використовує такі формули стереометрії:

### 1. Циліндр
- **Об'єм:** $V = \pi r^2 h$
- **Площа повної поверхні:** $S = 2\pi r(r + h)$

### 2. Конус
- Твірна: $l = \sqrt{r^2 + h^2}$
- **Об'єм:** $V = \frac{1}{3}\pi r^2 h$
- **Площа повної поверхні:** $S = \pi r(r + l)$

### 3. Еліпсоїд обертання (включаючи Кулю)
Нехай $a = r$ (радіус по осях X і Z), а $c = \frac{h}{2}$ (піввісь по осі Y).
- **Об'єм:** $V = \frac{4}{3}\pi a^2 c$
- **Площа поверхні** обчислюється залежно від форми еліпсоїда:
  - Сфера ($a = c$): $S = 4\pi a^2$
  - Стиснутий еліпсоїд ($a > c$): $S = 2\pi a^2 \left(1 + \frac{1 - e^2}{e} \text{artanh}(e)\right)$, де ексцентриситет $e = \sqrt{1 - \frac{c^2}{a^2}}$
  - Витягнутий еліпсоїд ($a < c$): $S = 2\pi a^2 \left(1 + \frac{c}{a e} \arcsin(e)\right)$, де $e = \sqrt{1 - \frac{a^2}{c^2}}$

## Особливості A-Frame реалізації
Оскільки в A-Frame відсутній примітив еліпсоїда, його реалізовано шляхом використання примітиву `<a-sphere>` із застосуванням нерівномірного масштабування по осі Y (`scale: 1 scaleY 1`). 

Для анімації утворення фігур використовується властивість `thetaLength` (для циліндра і конуса) та `phiLength` (для сфери), що дозволяє малювати фігуру секторами під час зміни кута.
