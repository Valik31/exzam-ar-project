# VR-проєкт: Тіла обертання

## Про проєкт
Цей освітній VR-додаток розроблено для інтерактивного вивчення теми "Тіла обертання" в курсі геометрії. Додаток дозволяє візуалізувати процес утворення об'ємних фігур (циліндра, конуса, еліпсоїда) шляхом обертання плоских фігур навколо осі.

## Функціональність
- Динамічна зміна параметрів фігури (радіус, висота).
- Автоматичний розрахунок об'єму та площі поверхні.
- Керована анімація обертання плоскої фігури (від 0 до 360 градусів) для демонстрації процесу утворення тіла.

## Технологічний стек
- **A-Frame:** для створення 3D-сцени та VR-середовища.
- **JavaScript:** для математичних розрахунків та маніпуляцій з DOM.
- **HTML/CSS:** для створення зручного інтерфейсу користувача.

## Як запустити
1. Завантажте папку з проєктом на свій комп'ютер.
2. Відкрийте файл `index.html` у будь-якому сучасному браузері.
3. Для перегляду самої 3D-сцени натисніть кнопку "Запустити демо" або відкрийте файл `demo.html`.
